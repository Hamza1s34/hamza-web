# Hamza Shah — Portfolio

A minimal, performant, and privacy-focused portfolio website for [Hamza Shah](https://hamza1s34.github.io/), built with HTML, CSS, and Vanilla JavaScript. 

Designed with a "digital paper" aesthetic, prioritizing clean typography and readable content.

## Overview

- **Design Aesthetic:** Editorial, paper-like design with minimal monochrome color palette (`#0a0a0a` on `#fafafa`).
- **Performance:** Zero dependencies, lightweight vanilla JavaScript, and pure CSS for fluid animations and layouts.
- **Accessibility:** Fully semantic HTML structure, proper ARIA labels, and contrast-compliant colors.
- **AI-Ready:** Includes a `llms.txt` file and structured `resume.md` specifically formatted for AI web crawlers and large language models.
- **SEO Optimized:** Complete JSON-LD structured data and Open Graph tags.

## Project Structure

- `index.html` — The main portfolio layout and structure.
- `styles.css` — Responsive CSS architecture with CSS variables and minimal media queries.
- `script.js` — Lightweight interaction logic (contact form routing, scroll-reveal animations).
- `resume.md` — The canonical, structured source of truth for all career experience.
- `llms.txt` — An AI-readable summary and index of the site's content.
- `favicon.svg` — Minimal vector branding.

## Development

To run this site locally, simply serve the directory with any local web server. For example:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx serve .
```

Then open `http://localhost:8000` in your browser.
